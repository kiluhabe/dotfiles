# Repository Notes

This is a dotfiles repository. Top-level directories represent managed
dotfile resource groups, not necessarily application source packages.

When inspecting configuration for a tool, first look under the matching
top-level resource directory. For Codex configuration, check `codex/.codex/`
before assuming the repository has no Codex config.

The `agents/` directory manages shared coding-agent dotfiles. Do not place
repository-specific guidance there.
